メモ：配列を使ったコードの簡略化
【気づき】
最初は switch や if を使って、「1ならグー」「2ならチョキ」と毎回判定しようとしていた。
しかし、配列（hands）を辞書のように使うことで、その判定処理をまるごと省略できた。

【ポイント】

Before: ループの中で毎回 switch で数値を文字（グー・チョキ・パー）に変換する必要があった。
After: hands[数値] と書くだけで、対応する文字が一発で取り出せるようになった。
メリット: コードが短くなり、ロジックがシンプルになった。


【サンプルコード】
let hands = ["グー", "チョキ", "パー"]; // 0:グー, 1:チョキ, 2:パー

// ランダムな数値（0〜2）を作る
let a = Math.floor(Math.random() * 3);

// 配列を使えば、if/switchなしで文字に変換できる！
// hands[0] -> "グー", hands[1] -> "チョキ"...
aHand[k] = hands[a];
